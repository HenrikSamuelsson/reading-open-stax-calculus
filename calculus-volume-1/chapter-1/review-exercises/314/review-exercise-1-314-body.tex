\subsection*{Review Exercise 1.314}

\subsubsection*{Instruction}

State the domain and range of the function $ h = \frac{1}{x + 4} $.

\subsubsection*{Solution}

Since $\frac{1}{x + 4} $ is defined when the denominator is nonzero, the domain is $\{x \mid x \ne -4}$.
